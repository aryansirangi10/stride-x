# StrideX — Advanced Shoe Website

A responsive, client-side shoe storefront demo with product listing, search, filtering, product modal, and a cart persisted to localStorage.

Preview locally:

1. Open a terminal in this folder
2. Run a static server: `python -m http.server 8000` or `npx serve .`
3. Open http://localhost:8000 in your browser

Git:
- A local git repo has been initialized and an initial commit created.
- To push, provide a remote URL or create a GitHub repo and run:
  `git remote add origin <URL>`
  `git push -u origin main`
